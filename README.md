# Endless Runner Variant

A fast-paced endless runner game school project built with [Defold](https://defold.com/). for the duration of 2 weeks

## Overview

This project is a variant implementation of a classic endless runner game, mostly featuring dynamic obstacles and coin collection.

## Features

- Smooth character movement and controls
- Procedurally generated obstacles
- Lua-based game logic with Defold engine
- Collect coins to get higher score

## Getting Started

### Prerequisites

- [Defold](https://defold.com/) game engine installed

### Installation

1. Clone this repository
2. Open the project in Defold
3. Build and run the game
